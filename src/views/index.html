Products

SubVT

SubVT (Substrate Validator Toolkit) is a native mobile application for iOS and Android phones, tablets and wearables that provides validator node operators with tools that aid them in running their validators on Polkadot and Kusama relay chains.

Development of SubVT is proudly supported by the Kusama Treasury through on-chain proposals #99 and #129 .

subvt.io

SubVT Telegram Bot

A feature-rich notification and reporting Telegram bot for the validators of the Polkadot and Kusama relay chains.

This bot is an upgrade over the deprecated 1KV Bot, rewritten in Rust as part of the SubVT backend.

SubVT Telegram Bot was successfully developed and delivered under the support of the Web3 Foundation Grants Program .

Kusama Bot Polkadot Bot

ChainViz

ChainViz Alpha is a real-time 3D chain visualization web application for Kusama.

ChainViz is a work in progress, and the currently released alpha version visualizes in real-time the block production process, network status and validators statistics of the Kusama network.

chainviz.app

ChainSynth

ChainSynth Alpha is the first step in the creation of a generative audio-visual synthesizer working with real-time blockchain data.

In its alpha version it generates audio and visuals using real-time data from the Polkadot relay chain. Please view more details on the GitHub repository.

chainsynth.app
